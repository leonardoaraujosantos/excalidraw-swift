# @cyberdynecorp/excalidraw-editor

The pure editor state machine — tools, selection, transform, undo — ported from ExcalidrawEditor.

Part of the **[Excalidraw — Native Swift + TypeScript/Svelte port](https://github.com/leonardoaraujosantos/excalidraw-swift-web)**: a from-scratch TypeScript + Svelte 5 implementation of Excalidraw whose packages share a data model, `.excalidraw` file format, and a custom WebSocket collaboration protocol with a native Swift/SwiftUI twin.

## Install

```sh
npm install @cyberdynecorp/excalidraw-editor
```

ESM-only. See the [monorepo README](https://github.com/leonardoaraujosantos/excalidraw-swift-web#readme) for the full package list and usage.
